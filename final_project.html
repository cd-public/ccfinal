Final Project Guidelines

Course: Thinking Machines

Duration: 5 Weeks

Group Size: Up to 2 students

Project Overview

Create a comprehensive data analysis project using Python, culminating in an interactive HTML report or hosted webpage. The goal is to address meaningful questions using data analysis, visualization, and modeling.

Objectives

  • Data Acquisition: Obtain and preprocess a dataset relevant to a chosen topic.
  • Data Analysis: Perform exploratory data analysis (EDA) and apply statistical or machine learning models to answer questions about the data.
  • Visualization: Create and incorporate meaningful and interactive visualizations.
  • Reporting: Compile findings into a structured HTML report.
  • Collaboration: Develop teamwork and project management skills.

Project Requirements

1. Topic Selection

  • Choose a topic allowing meaningful analysis, using public or self-collected data.
  • Get instructor approval by the end of Week 1.

2. Data Analysis & Modeling

  • Clean data using pandas and NumPy.
  • Conduct EDA to uncover patterns and insights.
  • Apply at least one statistical or machine learning model to answer research questions.

3. Visualization

  • Use libraries like Matplotlib, Seaborn, or Plotly to create at least three visualizations.
  • Include interactive elements if possible.

4. Report Creation

  • Document work in Jupyter Notebooks and convert it into an HTML report.
  • Include explanations, code snippets, outputs, and visualizations.

5. Technical Implementation

  • Use Git and GitHub for version control.
  • Host the final report using GitHub Pages or a similar platform.
  • Ensure well-documented and clean code.

Timeline and Milestones

Week 1: Project Proposal (Due: End of Week 1)

Deliverable: 1-2 page proposal including topic, significance, data source, research questions, and tools to be used.

Week 2: Data Acquisition & Preliminary Analysis (Due: End of Week 2)

Deliverable: Preliminary dataset and initial EDA results (summary stats, visualizations).

Week 3: In-Depth Analysis & Visualization (Due: End of Week 3)

Deliverable: Draft of analysis with models, findings, and supporting visualizations.

Week 4: Draft Report & Peer Review (Due: Mid & End of Week 4)

Deliverable: Full draft HTML report for peer review and feedback.

Week 5: Final Report & Presentation (Due: End of Week 5)

Deliverables: Final HTML report and a 10-minute presentation.

Grading Rubric (100 Points Total)

Criteria Points
Project Proposal 10
Data Analysis & Modeling 30
Visualizations 20
Report Quality 20
Technical Implementation 10
Presentation 10

Collaboration and Tools

  • Schedule regular team meetings, divide responsibilities clearly, and resolve conflicts promptly.
  • Use Python libraries for data manipulation (pandas, NumPy), visualization (Matplotlib, Seaborn, Plotly), and modeling (scikit-learn, statsmodels).
  • Use Jupyter Notebooks for code integration and nbconvert/Jupyter Book for HTML reports.
  • Host the report on GitHub Pages, Netlify, or similar.

Suggested Data Sources

  • Open data repositories: Kaggle, UCI Machine Learning Repository, Data.gov.
  • APIs for data collection: Twitter API, OpenWeatherMap API.

The final project should demonstrate your ability to answer a meaningful question or hypothesis about your dataset through analysis, modeling, and visualization.
